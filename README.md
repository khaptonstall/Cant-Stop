#Can't Stop Game
A branch of the REU-CAAR a the University of Maryland.

###Requires:
SDL2
SDL2_image
SDL2_ttf

###To play Can't Stop:

(1) Clone or download/unzip our repository.

(2) From the command line, run:
```
make
./cantstop
```
